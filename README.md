# Office Management Tool

A full-stack web application for managing office tasks, users, expenses, notes, and activities. Built with React.js (frontend), PHP (backend), and MySQL (database).

## Features
- User authentication and role-based access
- Task assignment, tracking, and team collaboration
- Expense submission, approval, and reporting
- Notes management for users
- Admin dashboard for user and task management
- File uploads for tasks and expenses

## Technologies Used
- React.js (Frontend)
- PHP (Backend API)
- MySQL (Database)

## Folder Structure
- `backend/` - PHP API and config files
- `public/` - Static assets and HTML
- `src/` - React components, pages, and services

## Setup Instructions
1. Clone the repository.
2. Set up MySQL and import the provided schema.
3. Configure database credentials in `backend/config/database.php`.
4. Start the PHP server (e.g., XAMPP) and React development server.

## Usage
- Register or log in as a user or admin.
- Create, assign, and track tasks.
- Submit and manage expenses.
- Add and view notes.
